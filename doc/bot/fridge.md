### fridge.setting.temperature  调温度
可能的slots：

slot key|slot value
--------|----------
room|冷冻 冷藏 保鲜 速冻 变温 恒温
temp_set|温度设置，如“设为”
set_temp|设置温度值，如“1”
temp_down|温度变小
temp_down_size|温度变小幅度
temp_up|温度增加
temp_up_size|温度增加幅度
temp_negative|代表温度为负值，如“零下”


冷藏室的温度设置到零下1度
```javascript
{
  "domain": "fridge",
  "intent": "fridge.setting.temperature",
  "slots": {
    "room": "冷藏室",
    "temp_set": "设置",
    "set_temp": "1",
    "temp_negative": "零下"
  }
}
```
冷藏室的温度降低一点
```javascript
{
  "domain": "fridge",
  "intent": "fridge.setting.temperature",
  "slots": {
    "room": "冷藏室",
    "temp_down": "降低"
  }
}
```

### fridge.setting.room 冷冻、xx室控制

slot key|slot value
--------|----------
room| 冷冻 制冷 冷藏
open| 打开冰箱舱室，如“开启”
close | 关闭冰箱舱室，如“关闭”

关闭冷冻室
```javascript
{
  "domain": "fridge",
  "intent": "fridge.setting.room",
  "slots": {
    "close": "关闭",
    "room": "冷冻"
  }
}
```


### fridge.setting.mode 设置冰箱模式

slot key|slot value
--------|----------
